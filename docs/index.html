全員が講師

FASTPASSの基本は「教えあい」です。学んだことを共有したり、フィードバックを互いにやり取りすることでスキルアップを目指します。

来たれ初心者

FASTPASSは初心者を歓迎します。段階を踏んで最先端の内容まで進んでいきましょう。

アウトプット

学んだ内容を活かして、例えば論文執筆・サービスローンチまで進むこともできます。簡単ではないですが、ともに学びましょう。

FASTPASSとは

プログラミング・API開発・データサイエンスなどのスキルアップに集中的に取り組み、最短距離で成果を出す集団です。2020年春からの本格活動と、東京大学教養学部学友会へのサークル登録を目指しています。

活動内容(予定)と初期メンバー

FASTPASSでは、スキルアップのための基礎を自主ゼミ形式で学びます。基礎から１つ１つ積み上げていき、実践的なアウトプットにつなげます。例えば初期メンバーは、大学院・専門課程で研究を続ける当事者であり、Pythonによるデータ分析・論文執筆・サービスローンチの経験者です。ともに学んできましょう。

メンバー募集

FASTPASSでは、ともに学ぶ仲間を募集しています。大学1,2年生または高校生で、東京大学駒場Iキャンパスでの活動が可能な方が対象です。サークル選択・進学選択に迷っている大学1,2年生、地方や非進学校の出身で情報格差で周りに負けたくない方、進路や受験大学に迷っている中高生を特に歓迎します。ご興味のある方は下記フォームよりご連絡ください。
